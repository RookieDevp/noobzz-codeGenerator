# noobzz-codeGenerator
代码生成器
基于若依代码生成思路，抽离出一个独立的代码生成器

用处：
1、生成自定义代码（前后端、SQL等）
2、自定义代码生成模板
3、多数据源（暂支持MySQL）
4、代码通用配置
